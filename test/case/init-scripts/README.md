## 构建信息
本项目使用 `yyl` 进行构建

### 安装 yyl
```
npm install yyl -g
```

### 相关命令
```
# 本地构建
yyl watch --proxy

# 远程映射
yyl watch --proxy --remote

# 打包
yyl all --isCommit
```
